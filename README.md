# EPI Child Tracker — Android App
### Western Province Health Office, Zambia

Standalone Android app for tracking under-5 immunisation in Kaoma District.

## Download APK
👉 Go to the **[Releases](../../releases)** tab and download the latest `EPI-Child-Tracker.apk`

## Install on Android
1. Download `EPI-Child-Tracker.apk`
2. Send to health worker phones via WhatsApp or USB
3. On each phone: **Settings → Security → Allow unknown sources**
4. Open the APK file → tap **Install**

## Features
- Register children with Under 5 Card Number (PPDD-FF-YYYY-NNNN)
- Full Zambia EPI schedule — 24 doses across 11 visit groups
- BCG, OPV 0-3, Penta 1-3, PCV 1-3, Rotavirus 1-3, IPV 1-2
- Malaria (RTS,S) 1, 2, 3 and Booster
- Measles-Rubella 1 & 2, Vitamin A 1 & 2
- Village + NHC/Zone registration
- PIN security, auto-lock, activity log
- 100% offline — no internet needed
- Export CSV for DHIS2 or Excel

## APK Build
Built automatically by GitHub Actions on every push. No Android Studio needed.
